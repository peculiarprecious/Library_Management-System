# Library Management System
**Dotnet Cohort 5 - Week 5 Assessment**

## Description
A C# console application that manages a collection of books using Properties, Lists, and validation logic.

## Features
- Add new books (Title, Author, ISBN)
- View all books in the collection
- Search for books by Title
- Track check-out status

## How to Run
1. Clone the repository.
2. Open in VS Code or Visual Studio.
3. Run `dotnet run` in the terminal.

## Technologies Used
- C#
- .NET SDK
- Git/GitHub
